Controllers

Basic overview of the core controllers used by the theme.

Root (MainController)

mandatory, Defined in scripts/core/controllers/mainController.js

Tasks handled by the controller:

  • handling $theme service options
  • handling progress loader

The primary function of this controller is exposing the getLayoutOption() method to the template, as used in index.html.

Navigation

mandatory, Defined in scripts/core/controllers/navigationController.js

The NavigationController is responsible for handling the left sidebar navigation and the horizontal navigation. Maverick's main navigation menu is entirely JS-driven, and you define menu items in the $scope.menu array scope variable. Example:

    $scope.menu = [
        {
            label: 'Overview',
            iconClasses: '',
            separator: true
        },
        {
            label: 'Dashboard',
            iconClasses: 'glyphicon glyphicon-home',
            url: '#/'
        },
        // ...
    ];

Menu item options are:

  • label: the text label to be shown on the rendered menu
  • url: the menu item URL
  • iconClasses: the icon beside the text label (FontAwesome/GlyphIcons)
  • html: arbitrary HTML to be displayed to the right, typically a badge
  • hideOnHorizontal: set to true if the item should be hidden on horizontal menu (boolean)
  • separator: set to true if the item is a menu separator (boolean)
