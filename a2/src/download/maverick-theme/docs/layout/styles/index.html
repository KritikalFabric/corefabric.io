LESS

The theme is built using LESS. However you should precompile the files in CSS for improved performance in production.

Stylesheets

All the .less files are located in assets/less/ directory of the admin theme and are called via styles.less file. Maverick code and Bootstrap code are kept separate using the extend-*.files so that you can easily switch to a new Bootstrap version by overwriting the files at) assets/less/bootstrap/_.

Extend File
extend-common.less dropdown, list-groups, wells, close
extend-core.less scaffolding, type, code, grid, tables, forms, buttons
extend-misc.less alerts, thumbnails, labels, badges, progressbars, input groups, carousel...
extend-nav.less navs, button-groups, breadcrumbs, paginations, pager
extend-popovers.less modals, tooltips, popovers
Less File
variables.less Bootstrap's variables.less, with additional variables for Maverick
layout.less Handles the layout & left sidebar of the theme
topnav.less The topnav dropdowns
leftbar.less The left sidebar navigation and search
rightbar.less The right sidebar, widget area and chatbox
tabs.less The CSS for the inline tabs and accordions
panels.less Completely redone to support tabs
tiles.less The Tiles components
apps.less The CSS of all the modified plugins
timeline.less The CSS for Timeline apge
loaders.less CSS Loader animation
app-inbox.less The inbox app styles
panel-widgets.less Custom widget styles
colorschemes.less Colors for topnav and sidebar are assigned. The color values themeselves are in the_assets\less\colors\ folder
extend-mixins.less Mixins used in Maverick and in some cases modification of bootstrap's mixins.
helpers.less Helper classes to speed up development time
demo.less Some styles used in displaying the demo, but no need for a fresh install

A pre-compiled style.css is provided in case you are not comfortable with LESS.

Fundamental styles

You can take a look at Bootstrap 3.3 Documentation for an extensive how to on how to use the theme and style the elements. Since the theme is based on top of BS 3.3, all the examples are valid albit styled to fit Maverick.

A large part of the theme can be customized with the included varialbles.less file.
