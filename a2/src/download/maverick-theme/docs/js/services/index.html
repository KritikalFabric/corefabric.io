Services

Maverick ships with a set of important core services that can be injected into controllers as needed. These services are a part of the core theme services, available as the theme.core.services module. Here is a usage manual for the custom services.

$theme

Defined in scripts/core/services/theme.js

The $theme service is used for controlling theme-wide behaviors and appearance. It is intended to be injected into controllers as needed, as used in scripts/core/controllers/mainController.js.

Usage example

app.controller('MyController', ['$theme', function ($theme) {
    ...
}]);

Methods

It primarily exposes a getter and a setter for various theme options.

$theme.get('optionName');

And,

$theme.set('optionName', value);

The options are as follows:

  • fixedHeader: header is fixed (boolean, default true),
  • headerBarHidden: pull-down header bar is hidden (boolean, default true),
  • leftbarCollapsed: left sidebar is collapsed (boolean, default false),
  • leftbarShown: left sidebar is hidden (boolean, default false),
  • rightbarCollapsed: right infobar is collapsed (boolean, default false),
  • fullscreen: page is fullscreen (boolean, default false),
  • layoutHorizontal: horizontal layout is enabled (boolean, default false),
  • layoutHorizontalLargeIcons: horizontal layout uses large icons (boolean, default false),
  • layoutBoxed: boxed layout is enabled (boolean, default false),
  • showSmallSearchBar: (boolean, default false),
  • topNavThemeClass: nav bar class, default navbar-default,
  • sidebarThemeClass: left sidebar class, default sidebar-midnightblue,
  • showChatBox: chat box is visible, (boolean, default false),
  • pageTransitionStyle: page transition style, one of the styles from animate.css, default fadeIn,
  • dropdownTransitionStyle: dropdown animation style, default flipInX

In addition, there is a getBrandColor() method you can use to retrieve a brand color. Example:

$theme.getBrandColor('primary');

Events

Sometimes you might wish to subscribe to changes in the layout. For this the $theme service emits 3 scope events which are as follows:

  • themeEvent:maxWidth767: page max width is now below 768px
  • themeEvent:changed:<option_name>: theme option has been changed (passes new value to listeners)
  • themeEvent:changed: a theme option has been changed (passes { key: <option_name>, value: <option_value>} to listeners)

progressLoader

Defined in scripts/core/services/services.js

The progressLoader is wrapper around the Skylo progress bar.

Methods

The factory has the following methods:

  • start: start loader
  • end: finish loading
  • set: set progress, takes an integer parameter as percentage of progress
  • inch: inch ahead progress, takes an integer parameter as percentage of progress to increment

lazyLoad

Defined in scripts/core/services/services.js

The lazyLoad service is a delayed script loader to be used in route definitions. Example:

app.config(['$provide', '$routeProvider', function ($provide, $routeProvider) {
    $routeProvider
    .when('/', {
        templateUrl: 'views/index.html',
        resolve: {
            lazyLoad: ['lazyLoad', function (lazyLoad) {
                return lazyLoad.load([
                    'bower_components/fullcalendar/fullcalendar.js',
                ]);
            }]
        }
    })
    // other definitions
    .otherwise({
        redirectTo: '/'
    });
}]);
