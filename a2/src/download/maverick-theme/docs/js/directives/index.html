Directives

Maverick's custom directives and how to use them.

Panels

You can create panels quickly and painlessly using the panel element. Example:

<panel panel-class="panel-warning" heading="Draggable panel" ng-drag="true">
    <!-- Panel content goes here -->
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>
</panel>

Panel options are passed through attributes on the panel element. The options are:

  • panel-class: one of many panel-* classes, * being one of the brand colors
  • heading: panel title/heading
  • ng-drag: whether or not the pannel is draggable (boolean, default false)

Panel Controls

Panels can have one of many widget controls declared within panel > panel-controls. Example:

<panel panel-class="panel-warning" heading="Drag This" ng-drag="true">
    <panel-controls>
        <panel-control-collapse></panel-control-collapse>
        <panel-control-title></panel-control-title>
        <panel-control-colors></panel-control-colors>
        <panel-control-refresh type="circular" is-loading="isLoadingPanel" ng-click="refreshAction()"></panel-control-refresh>
        <button class="button-icon"><i class="glyphicon glyphicon-cog"></i></button>
        <button class="button-icon"><i class="glyphicon glyphicon-wrench"></i></button> 
    </panel-controls>

    <!-- Panel content goes here -->
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>
</panel>

They can be arbitrary HTML control elements, or one of the following special directives:

  • panel-control-collapse: adds a collapse/expand control to panel
  • panel-control-title: adds a panel title editor control to panel
  • panel-control-colors: adds a panel color picker control to panel (changes the panel's class)
  • panel-control-refresh: adds a refresh control to the panel

More on panel-control-refresh

Options can be passed as attributes of the panel-control-refresh directive. type can be one of circular, dots or clock. The is-loading attribute is used to pass a boolean scope variable that tells the panel whether or not to activate loading animation. You can pass in a custom refresh method through the ng-click directive which references a scope method.

Tiles

There are 4 directives for tiles for different use cases. Examples:

In controller:

$scope.tileDefLarge = { title: 'Page Views', href: '#/ui-tiles', titleBarInfo: '-7.6%', text: '1.6K', color: 'info', classes: 'fa fa-eye' };
$scope.tileDefShortcut = { text: 'Page Views', href: '#/ui-tiles', titleBarInfo: '7', color: 'info', classes: 'fa fa-eye' };
$scope.tileDefSimple = { title: 'Orders', href: '#/ui-tiles',text: '9', color: 'danger'};

In HTML:

<tile-large data="tileDefLarge"></tile-large>
<tile-shortcut data="tileDefShortcut"></tile-shortcut>
<tile-simple data="tileDefSimple"></tile-simple>
<tile type="default" heading="Disk Space">
    <!-- arbitrary content goes here -->
</tile>

Note The tile directive doesn't take a data attribute for options, but it does take a heading attribute for title. The type (color) of the tile can be specified with the type attribute. The inner content can be arbitrary.

Flot Charts

Flot charts can be using flot-data and flot-options directives. Example:

In controller:

  $scope.plotStatsData = [
    {
      data: [
          [1, 1100],
          [2, 1400],
          [3, 1200],
          [4, 800],
          [5, 600],
          [6, 800],
          [7, 700],
          [8, 900],
          [9, 1700],
          [10, 1200],
      ],
      label: "Page Views"
    },
    {
      data: [
          [1, 3100],
          [2, 4400],
          [3, 900],
          [4, 1800],
          [5, 2600],
          [6, 3800],
          [7, 1700],
          [8, 2900],
          [9, 900],
          [10, 2200],
      ],
      label: "Unique Views"
    }
  ];

  $scope.plotStatsOptions =  {
      series: {
          stack: true,
          lines: {
              show: true,
              lineWidth: 2,
              fill: 0.1
          },
          splines: {
            show: true,
            tension: 0.3,
            lineWidth: 3
          },
          points: {
              show: true
          },
          shadowSize: 0
      },
      grid: {
          labelMargin: 10,
          hoverable: true,
          clickable: true,
          borderWidth: 0
      },
      tooltip: true,
      tooltipOpts: {
        defaultTheme: false,
        content: "View Count: %y"
      },
      colors: ["#b3bcc7"],
      xaxis: {
          tickColor: 'rgba(0,0,0,0.04)',
          ticks: 10,
          tickDecimals: 0,
          autoscaleMargin: 0,
          font: {
              color: 'rgba(0,0,0,0.4)',
              size: 11
          }
      },
      yaxis: {
          tickColor: 'transparent',
          ticks: 4,
          tickDecimals: 0,
          font: {
              color: 'rgba(0,0,0,0.4)',
              size: 11
          },
          tickFormatter: function (val, axis) {
              if (val>999) {return (val/1000) + "K";} else {return val;}
          }
      },
      legend : {
          labelBoxBorderColor: 'transparent',
      }
  };

In HTML:

    <div class="" flot-chart style="width: 100%; height:320px"
        data-flot-data="plotStatsData"
        data-flot-options="plotStatsOptions"
    ></div>

Morris Charts

Invoke Morris charts using the svg-chart and data-type directives.

In controller:

  $scope.lineChart = {
    data: [
        { y: '2006', a: 100, b: 90 },
        { y: '2007', a: 75,  b: 65 },
        { y: '2008', a: 50,  b: 40 },
        { y: '2009', a: 75,  b: 65 },
        { y: '2010', a: 50,  b: 40 },
        { y: '2011', a: 75,  b: 65 },
        { y: '2012', a: 100, b: 90 }
    ],
    xkey: 'y',
    ykeys: ['a', 'b'],
    labels: ['Series A', 'Series B'],
    lineColors: [$theme.getBrandColor('inverse'),$theme.getBrandColor('midnightblue')]
  };

In HTML:

<div svg-chart="lineChart" data-type="'Line'"></div>

Canvas Charts

Use the canvas-chart and data-type directives for creating Canvas charts.

In controller:

  $scope.lineChartData = {
    labels : ["January","February","March","April","May","June","July"],
    datasets : [
      {
          fillColor : "rgba(220,220,220,0.5)",
          strokeColor : "rgba(220,220,220,1)",
          pointColor : "rgba(220,220,220,1)",
          pointStrokeColor : "#fff",
          data : [65,59,90,81,56,55,40]
      }, {
          fillColor : "rgba(151,187,205,0.5)",
          strokeColor : "rgba(151,187,205,1)",
          pointColor : "rgba(151,187,205,1)",
          pointStrokeColor : "#fff",
          data : [28,48,40,19,96,27,100]
      }
    ]
  };

In HTML:

<canvas height="300" width="600" canvas-chart="lineChartData" data-type="'Line'"></canvas>

Misc Directives

animatePageContent

Defined in scripts/core/directives/ui.js

Used to animate page content specified by .container-fluid .animated-content element of the currently loaded view.

slideOutNav

Defined in scripts/core/directives/ui.js

Used on the left sidebar menu. Toggles slide on menu items list based on whether or not its parent is marked as open.

toTopOnLoad

Defined in scripts/core/directives/ui.js

A simple directive that scrolls the page to top when a page loads.

backToTop

Defined in scripts/core/directives/ui.js

Scrolls page to top on page load.

positionChatBox

Defined in scripts/core/directives/ui.js

Positions the floating chat box on the right infobar.
