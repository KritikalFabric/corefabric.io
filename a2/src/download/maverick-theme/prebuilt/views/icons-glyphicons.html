One Font, 180 Icons

In a single collection, Glyphicons is a pictographic language of web-related actions.

CSS Control

Easily style icon color, size, shadow, and anything that's possible with CSS.

Infinite Scalability

Scalable vector graphics means every icon looks awesome at any size.

Desktop Friendly

To use on the desktop or for a complete set of vectors, check out the cheatsheet provided below.

Perfect on Retina Displays

Glyphicons icons are vectors, which mean they're gorgeous on high-resolution displays.

Screen Reader Compatible

Glyphicons won't trip up screen readers, unlike other icon fonts.
Heads up! For performance reasons, all icons require a base class of glyphicon as well as individual icon classes.
