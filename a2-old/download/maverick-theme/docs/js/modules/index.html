Core Modules

Maverick's core modules are needed for the theme to work.

Panels

mandatory, Defined in scripts/core/modules/panels/panels.js

Maverick's advaned UI panels are handled with this module.

Templates

mandatory, Defined in scripts/core/modules/tempaltes.js

To speed up the process of loading templates in Angular, Maverick caches some of the most frequently used HTML templates with Angular's $templateCache service.

Essentially these are JS versions of HTML snippets in views/templates folder. The templates.js file is generated via grunt by running the grunt ngtemplates command. If you're wondering after changing the HTML files why the changes aren't showing on the theme, you need to run the grunt command.

Template Overrides

mandatory, Defined in scripts/core/modules/tempalteOverrides.js

Enhancements to third-party templates and modules are done in this module.
