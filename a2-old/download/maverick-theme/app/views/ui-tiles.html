Maverick gives you the ability to create tiles programmatically using the tile-large directive.

Tiles can also have arbitrary content if the text property is not defined (2nd example).

NOTE If you specify a link in the href attribute, the tile functions as a link.

<tile-large data="tileDef"></tile-large>
<tile-large data="tileDef2"><span sparklines values="5,4,7,6,9,5,8,2,6,4,6,7"></span></tile-large>

// inside the controller
$scope.tileDef = { title: 'Page Views', href: '#/ui-tiles', titleBarInfo: '-7.6%', text: '1.6K', color: 'info', classes: 'fa fa-eye' };
$scope.tileDef2 = { title: 'Page Views', href: '#/ui-tiles', titleBarInfo: '-7.6%', color: 'info', classes: 'fa fa-eye' };
